# Эффективная теоря поля

В Стандартной модели существуют **сильно разнесённые масштабы энергий**:

- электрослабый масштаб  \( M_W \sim 80\ \text{GeV} \)
- адронный (QCD) масштаб \( \Lambda_{\text{QCD}} \sim \mathcal O(1)\ \text{GeV} \)

При энергиях \(E \ll M_W\) тяжёлые степени свободы (**\(W, Z, t, H\)**)  
**не могут рождаться на массе**, но продолжают влиять на физику через виртуальные эффекты.

Идея EFT — **описать эту физику без явного упоминания тяжёлых частиц**.

Проиллюстрируем идеи EFT на примере теории с 2 скалярными полями

## EFT: скалярная теория

Рассмотрим модель с двумя скалярными полями:

- лёгкое: \(\phi\) с массой \(m \ll M\)
- тяжёлое: \(\Phi\) с массой \(M\)

Полный лагранжиан:

$$
\mathcal L =
\frac{1}{2}(\partial \phi)^2
+ \frac{1}{2}(\partial \Phi)^2
- \frac{1}{2} m^2 \phi^2
- \frac{1}{2} M^2 \Phi^2
- \frac{b}{2}\,\phi^2 \Phi
$$

При энергиях \(E \ll M\) поле \(\Phi\) не является динамическим.  
Его можно исключить, решив уравнение движения:

$$
\Phi = \frac{b}{2} \frac{1}{-\square-M^2}\phi^2 = -\frac{b}{2M^2}\phi^2 + O(\frac{\square}{M^2})
$$

Подставляя обратно, получаем **эффективный лагранжиан** (1):
{ .annotate }

1.  
    - Точки означают операторы более высокой размерности
    - Они подавлены степенями \(1/M\)

\begin{equation}
  \mathcal L_{\text{eff}} =
  \frac{1}{2}(\partial \phi)^2
  - \frac{1}{2} m^2 \phi^2
  + \frac{b^2}{8M^2}\,\phi^4
  + \dots
  \label{eq:efflag}
\end{equation}

В EFT вся динамика тяжелого поля \(\Phi\) заменяется **точечными взаимодействиями**, которые нужно сшить с полной теорией:

\begin{equation}
  \mathcal L_{\rm EFT} \supset  - \frac{1}{4!} C_{(4,0)} \, (\phi^4) - \frac{1}{6!} C_{(6,0)} \, (\phi^2 \partial^2 \phi^2) + \cdots
  \label{eq:EFLag}
\end{equation}

Сравнивая с EFT [@](#eq:EFLag){data-preview} , видим, что ведущий оператор \(\phi^4\) имеет **коэффициент сшивки**:

\[
  C_{(4,0)} = -\frac{3 b^2}{M^2} = -4!\frac{ b^2}{8M^2}
\]

(Это cовпадает с ожидаемым коэффициентом из [@](#eq:efflag){data-preview} )

Процедура сшивки  **фиксирует коэффициенты операторов EFT на некотором масштабе** \(\mu \sim M\), где M — масса тяжелого поля (например, \(\Phi\) или W-бозона).  

Общий алгоритм такой:

1. Вычисляем амплитуды в полной теории
2. Раскладываем их по степеням \(1/M^2\) при энергиях \(E \ll M\)
3. Сопоставляем с теми же амплитудами в EFT, определяя коэффициенты \(C_{(n,m)}(\mu=M)\)

### Зачем нужна EFT?

Если в процессах появляются большие логарифмы или нам нужно спустить теорию на масштаб \(E\)

\[
  \ln \frac{M^2}{E^2} \gtrsim 1,
\]

то простое вычисление амплитуды будет неточным: нужно использовать **ренормгрупповые уравнения** для коэффициентов операторов \(C_{(n,m)}(\mu)\). В полной теории такую ресуммацию сделать не получится, так как есть масштаб \(M\).
