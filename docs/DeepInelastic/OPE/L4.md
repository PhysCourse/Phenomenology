# Операторное Разложение (OPE)

Вспомним из прошлых [лекций](../../KaonDecay/FCNC/L2.md){ref-require} то, как мы получали эффективную теорию.

У нас был лагранжиан 

$$
\mathcal L =
\frac{1}{2}(\partial \phi)^2
+ \frac{1}{2}(\partial \Phi)^2
- \frac{1}{2} m^2 \phi^2
- \frac{1}{2} M^2 \Phi^2
- \frac{b}{2}\,\phi^2 \Phi
$$

где $m \ll M$. В результате, при масштабах импульса $\mu \ll M$, влияние поля $\Phi$ на физику проявляется лишь в виде набора локальных операторов эффективной теории для поля $\phi$.

Так, простейшая амплитуда $2 \to 2$ имела значение

\begin{equation*}
    i\mathcal{A} = \sum_{p^2=s,t,u} \cfrac{-3i b^2}{p^2 - M^2} \approx 3i \frac{b^2}{M^2}
\end{equation*}

При $p^2 \ll M^2$ вместо двух точек $x,y$ и пропагатора поля $\Phi$, $G(x-y)$, у нас теперь одна вершина $x$.

Таким образом, всё, что происходит на расстояниях $x-y \sim \frac{1}{M}$, не видно на масштабе $\mu$ и описывается локальными операторами.

Теперь мы будем действовать в рамках одной теории, но суть останется похожей.

---

## Определение OPE

Рассмотрим некоторое число локальных операторов (мы ограничимся двумя): $O_1(x)$ и $O_2(y)$ и всевозможные матричные элементы, в которых участвует их произведение

\[
    G_{12}(x,y,x_1,\dots,x_n) = \langle \Omega| T O_1(x)O_2(y) \phi_1(x_1) \dots \phi_n(x_n) |\Omega \rangle
    \label{eq:O12_green_function}
\]

Эта область описывается условием $x-y \ll x_i - x_j$.
В этом случае имеется масштаб $\mu^{-1} \sim x_i-x_j$, причём $x-y \ll \mu^{-1}$, и возникает предположение о разложении произведения операторов (Operator Product Expansion):

\[
    \boxed{
     O_2(y) = \sum_{n} O_n(y) C_n(x-y)
    }
    \label{eq:OPE_formula}
\]

Под этим разложением имеется в виду равенство функций Грина

\[
    \begin{split}
    G_{12}(x,y,\{x_i\}) = \langle \Omega| T O_1(x)O_2(y) \phi_1(x_1) \dots \phi_n(x_n) |\Omega \rangle = \\
    = \sum_{n} C_n(x-y) \langle \Omega| T O_n(y) \phi_1(x_1) \dots \phi_n(x_n) |\Omega \rangle = \\
    = \sum_{n} C_n(x-y) G_{n}(y,\{x_i\})
    \end{split}
    \label{eq:OPE_formula_what}
\]

Причём равенство [@](#eq:OPE_formula) имеет {==операторный==} характер: равенство функций Грина [@](#eq:OPE_formula_what)
выполняется для любого набора полей $\phi_i(x_i)$.

!!! warning "Замечание"
    Отметим, что локальные операторы $O_1(x)$, $O_2(y)$, $O_n(y)$ должны быть перенормированы на некотором масштабе $M$.
    Об этом см. [лекцию](../../KaonDecay/FCNC/L3.md#eq:RenormLocOper){ref-require}.

    \[
        O_I(x) = [O_I(x)]_M = \sum_J (Z^{-1})_{IJ}(M) O^0_J   
    \]

    Функции $C_n(x-y)$ зависят от масштаба $M$. Для них можно записать уравнение Каллана—Симанчика 
    (оно получается вычитанием соответствующих уравнений [@](../../KaonDecay/FCNC/L3.md#eq:KallanSimanchic) для функций Грина $G_{12}$ и $G_n$).

\[
    \left[ \frac{\partial}{\partial\ln M} + \beta \frac{\partial}{\partial g} + \gamma_1 + \gamma_2 - \gamma_n \right] C_n(x-y,M)
\]

Поскольку $C_n(x-y,M)$ зависит только от двух размерных параметров, можно написать его в виде

\[
    C_n(x,M) = \left(x^2 \right)^{d_n-d_1-d_2} \tilde{C}(Mx)
\]

где $d_1, d_2, d_n$ – размерности соответствующих операторов, а $\tilde{C}(Mx)$ – безразмерная функция.

Решение представляется в виде:

\[
    C_n(x,M) = \left(x^2 \right)^{d_n-d_1-d_2} c(g(M_x)) \exp \left[ \int_{M_x}^{M}{ (\gamma_n - \gamma_1 - \gamma_2) d \ln M' }\right] 
\]

где $M_x$ – масштаб, равный $1/|x|$.

В КХД, где аномальные размерности и $\beta$-функция имеют вид:

\[
    \gamma_n = -a_n \frac{g^2}{16\pi^2} \quad \beta = -b_0 \frac{\alpha_S^2}{2\pi}    
\]

Мы можем написать решение в виде:

\[
    C_n(x) \sim  \left(x^2 \right)^{d_n-d_1-d_2} \left(\frac{\ln (|x|^2\Lambda^2)^{-1}}{ \ln(M^2/\Lambda^2)} \right)^{(a_n-a_1-a_2)/b_0}
\]

В случае, если операторы смешиваются, нужно рассматривать разложение для произведения операторов:

\[
    O_m(x) O_k(y) = \sum_N{ C^N_{mk}(x-y) O_N(y)}
\]

Тогда уравнение Калана-Симанчика имеет вид:

\[
    \left[ \frac{\partial}{\partial\ln M} + \beta \frac{\partial}{\partial g}\right]  C^N_{mk}(x,M) + 
    \gamma_{mm'} C^N_{m'k}(x,M) + \gamma_{kk'} C^N_{mk'}(x,M) - \gamma_{N'N} C^{N'}_{mk} = 0 
\]

где сумма идет по повторяющимся штрихованным индексам.

Выражания также удобно иметь в импульсном представлении. В наших обозначениях 

\[
    G(p,p_y,\{p_i\}) (2\pi)^4 \delta^{(4)}(p_y +p_1 + \dots p_n)= \int {d^4(x-y) d^4y e^{ip (x-y)} e^{ip_y y} e^{i \sum_i p_i x_i} G(x,y,\{x_i\}) }
\]

Таким образом, особенности при $x \to 0$ переходят в особенности в импульсном пространстве при $q \to \infty$.

\[
    \sim x^{-n} \text{при} x\to 0 \Rightarrow \sim q^{n-4} \text{при} q \to \infty
\]

## Пример вычисления OPE $\phi(x)\phi(y)$ 

**Алгоритм** действия такой же как и в EFT. Для разложения нам нужно

- Написать все возможные локальные операторы, позволенные симметриями
- Шаг за шагом по теории возмущений сшивать функции Грина $G_{12}(x,y,\{x_i\})$ и суммы $C_n(x-y) G_n(y,\{x_i\})$

Рассмотрим операторное произведение $O_1(x) = \phi(x)$, $O_2(x) = \phi(y)$ в теории $\lambda \phi^4/4!$.

В свободной теории операторное разложение сводится к теореме Вика

\[
    \phi(x) \phi(y) = D_F(x-y) + :\phi^2(y): + \sum_{n} \frac{1}{n!}(x-y)^{\mu_1}\dots (x-y)^{\mu_n}: [\partial_{\mu_1}\dots\partial_{\mu_n}\phi(y)]\phi(y):
    \label{eq:OPE_free}
\]

где под знаком $:ab:$ понимается нормальное упорядочивание, исключающее внутрипетлевые вклады.

- Первое слагаемое возникает из функции Грина без полей, это точный результат:

\[
    \langle 0 | T \phi(x) \phi(y) | 0 \rangle = D_F(x-y)
\]

- Второе согласовано с функциями Грина с 2 полями $\phi(x_1)$, $\phi(x_2)$ (рисунок)

\[
    \langle 0 | T \phi(x) \phi(y) \phi(x_1) \phi(x_2)| 0 \rangle = 
    D_F(x-x_1) D_F(y-x_2) + (x_1 \leftrightarrow x_2)
\]

### 1-й порядок теории возмущения.

В первом порядке по $\lambda$ мы должны учесть диаграммы с одной вершиной $\lambda \phi^4/4!$.

По определению пропагатор равен:

\[
    D_F(x-y) = \int \operatorname{d}\!\Pi_p \frac{i e^{-ip(x-y)}}{p^2 + i\epsilon} \quad 
    \text{где} \quad \operatorname{d}\!\Pi_p = \frac{d^4p}{(2\pi)^4}
    \label{eq:propagator_momentum}
\]

Мы преобразуем выражение для первой поправки и увидим, 
что результат будет пропорционален $D_F(y-x_1)\cdot D_F(y-x_2)$ – внешние ноги, 
которые мы всегда можем ампутировать.

??? success "Преобразования"

    [Диаграмма](#fig:NLO_phiphi_lambda) равна

    \[
        G = (-i\lambda) \int{ d^4 z D_F(x-z) D_F(z-y) D_F(z-x_1) D_F(z-x_2)}
    \]

    Подставляем для всех пропагаторов их представление в виде интеграла по импульсам [@](#eq:propagator_momentum):

    \[
        (-i\lambda) \int{ d^4 z \operatorname{d}\!\Pi_q \operatorname{d}\!\Pi_p \operatorname{d}\!\Pi_{p_1} \operatorname{d}\!\Pi_{p_2} 
        \frac{i e^{-iq(x-z)}}{q^2 } 
        \frac{i e^{-ip(z-y)}}{p^2 } 
        \frac{i e^{-ip_1(z-x_1)}}{p_1^2 } 
        \frac{i e^{-ip_2(z-x_2)}}{p_2^2 }
        }
    \]

    Интеграл по $z$ дает нам сохранение импульса, а интеграл по $p$ снимет дельту-функцию, так что мы получаем

    \[
        i\lambda \int{
            \operatorname{d}\!\Pi_q 
            \frac{ e^{-iq(x-y)} }{q^2 p^2}   
            \operatorname{d}\!\Pi_{p_1} \frac{ i e^{-ip_1(y-x_1)}}{p_1^2} 
            \operatorname{d}\!\Pi_{p_2} \frac{i e^{-ip_2(y-x_2)}}{p_2^2}
        }
    \]

    Последние два интеграла – это пропагаторы внешних ног, которые мы ампутируем.

    Импульс $p$ равен $p = q -p_1 - p_2  = q-P$, так что знаменатель равен $q^2(q-P)^2$

    В результате мы получаем ответ

\[
    G_{\text{амп}}(x-y) = i\lambda \int{
    \frac{d^4 q}{(2\pi)^4} 
    \frac{ e^{-iq(x-y)} }{(q^2 + i\epsilon) ((q-P)^2 + i\epsilon)}  
    }
    \label{eq:OPE_lambda_integral}
\]

С помощью фейнмановского параметра мы можем легко написать:

\[
    \begin{split}
    G_{\text{амп}}(x-y) = i\lambda  \int{
    \frac{d^4 q}{(2\pi)^4} 
    \frac{ e^{-iq(x-y)} e^{-i\xi P(x-y)} }{(q^2 - \Delta^2 + i\epsilon)^2} d\xi 
    } \\
    \text{где} \quad \Delta^2 = -P^2 \xi(1-\xi)
    \end{split}
    \label{eq:OPE_pheynman_parameter}
\]

> Будем работать в пространственно-подобных координатах и импульсах.

так что $P^2 < 0$ и $\Delta^2 > 0$, а $(x-y)^2 < 0$. Это удобно по нескольким причинам:

- Мы можем перейти в евклидовы координаты, заменив $p^0$ на $ip^0_E$ и $x^0$ на $ix^0_E$.
- В пространстве Минковского есть области, где импульсы велики, а их квадраты малы, что приводит к дополнительной иерархии масштабов и появлению полюсов и разрезов в комплексной плоскости.

Поскольку пространство однородно, можно положить $y=0$.

\[
    G_{E}(x_E) = -\lambda  \int{ 
    \frac{d^4 q_E}{(2\pi)^4} 
    \frac{ e^{iq_E x_E} }{(q_E^2 + \Delta^2)^2} e^{i\xi P_E x_E} d\xi 
    } 
    \label{eq:OPE_Euclid}
\]

#### Оценка областей интеграла

Прежде чем что-то вычислять, полезно оценить, какие области привносят расходимости при $x_E \to 0$.

- Область интеграла при $q_E \lesssim  1/x_E$: в этом случае по порядку величины $e^{iq_E x_E} \approx 1$ и интеграл равен:

    \[
        \int_0^{1/x_E} \frac{d^4 q_E}{(2\pi)^4} \frac{1}{(q_E^2 + \Delta^2)^2} = - C \frac{\lambda}{16\pi^2} \ln (\Delta^2 x_E^2)
    \]

    Это и есть ожидаемый логарифмический вклад в коэффициентную функцию $C_n(x)$.

- При $q_E >  1/x_E$ можно сделать замену $u = q |x_E|$ и получить конечную величину:

\[
    \sim \int_{1}^{\infty} \frac{e^{iu}}{u} du =\text{const}
\]

- Теперь рассмотрим матричный элемент с оператором $:\phi^2(0):$ от средней [диаграммы](#fig:NLO_phiphi_lambda). 
Будет такое же выражение, но с вместо $x_E^2$ будет  $1/\Lambda^2$
(можно сказать, что матричный элемент равен пределу при $x_E \to 0$, т.е. пределу $\Lambda \to \infty$ импульса обрезания).
  

    \[
        G_{\phi^2}(\Lambda) = -  C \frac{\lambda}{16\pi^2} \ln \left(\frac{\Delta^2}{\Lambda^2}\right) + \dots
    \]

    Если к этому выражению добавить контрчлен перенормировки оператора $:\phi^2(0):$ на масштабе $M$
    (третья [диаграмма](#fig:NLO_phiphi_lambda) ), то мы получим конечный результат: 
    
    
    \[
        G_{\phi^2_R}(M) = -  C \frac{\lambda}{16\pi^2} \ln \left(\frac{\Delta^2}{M^2}\right) + \dots
    \]

    Если рассмотреть матричные элементы диаграмм [первого](#fig:LO_phiphi) и [второго](#fig:NLO_phiphi_lambda) порядка, то ампутированная левая часть имеет вид

    \[
        \text{левая часть} = 1 - C \frac{\lambda}{16\pi^2} \ln (\Delta^2 x_E^2) + \dots
    \]

    Правая часть равна:

    \[
        \text{правая часть} = 1 - C \frac{\lambda}{16\pi^2} \ln \left(\frac{\Delta^2}{M^2}\right) + \dots
    \]

    Таким образом, для сшивки правой и левой частей в выражении [@](#eq:OPE_formula) нужно ввести коэффициентную функцию

    \[
        C_{\phi^2_R}(x) = 1 - C \frac{\lambda}{16\pi^2} \ln (M^2 x_E^2) + \dots
    \]


#### Честное вычисление (левая часть)

Итак, у нас есть интеграл [@](#eq:OPE_Euclid), который мы хотим вычислить Для этого воспользуемся формулой 
[@](../../Appendix/Formulas.md#eq:propagator_position_space) при $n=2$ в пределе $\epsilon \to 0$.

\[
    G_{E}(x_E) = -\frac{\lambda}{16\pi^2} \int{d\xi \left(\frac{y}{2}\right)^{\epsilon} 2K_{\epsilon}(y) e^{i\xi P_E x_E}} \quad \text{где} \quad y^2 = \Delta^2 x_E^2
    \label{eq:OPE_Euclid_final}
\]

??? info "про функцию Бесселя"

    При $y \to 0$ можно написать ряд:

    \[
        2 K_{\epsilon}(y) = \frac{\pi} {\sin{\pi\epsilon}} \sum_k{ \frac{(y^2/4)^k}{k!} \left( \frac{ (y/2)^{-\epsilon} }{\Gamma(1-\epsilon+k)} - \frac{ (y/2)^{\epsilon} }{\Gamma(1+\epsilon+k)} \right)}
    \]

    В пределе $\epsilon \to 0$ мы получаем сходящийся ряд

    \[
        2 K_{0}(y) = \sum_k{ \frac{(y^2/4)^k}{k!^2} \left( -\ln{(y^2/4)} + 2\psi(k+1) \right)}
    \]

    Напомним также, что $\psi(k+1)$ — это дигамма-функция (производная логарифма гамма-функции):

    \[
        \psi(k+1) = -\gamma + H_n = -\gamma + \sum_{m=1}^k \frac{1}{m}
    \]


В результате мы получаем следующее разложение полной ампутированной функции Грина с учетом [диаграмм](#fig:LO_phiphi) LO:

\[
    \begin{split}
    &G^{\text{полн}}_{E}(x_E) = e^{ip_{1E}x_E} + e^{ip_{2E}x_E} - \\
    &- \frac{\lambda}{16\pi^2} \int{d\xi e^{i\xi P_E x_E} \sum_k{ \frac{(y^2/4)^k}{k!^2} \left( -\ln{(y^2/4)} + 2\psi(k+1) \right)}}
    \end{split}
    \label{eq:OPE_Euclid_final_approx}
\]

!!! success "Структура разложения"
    - Если разложить экспоненту, то выражение будет содержать суммы вида

    \[
        y_E^{\mu_1}\dots y_E^{\mu_n} \ln (y^2) \quad \text{где} y_E = \Delta x_E
    \]
    
    - Если мы рассмотрим матричные элементы перенормированных операторов с коэффициентами вида 
    
        $$
            :\partial_{\mu_1}\dots \partial_{\mu_n} \phi(0) \phi(0): x_E^{\mu_1}\dots x_E^{\mu_n}
        $$

        то мы получим аналогичные члены, за исключением того, что в них логарифм будет заменён на $\ln (\Delta^2/M^2)$,
        что позволяет однозначно определить коэффициентные функции для операторов, появляющихся в OPE.

??? info "Еще раз про перенормировку операторов"

    Из выражений выше возникает желание определить перенормированный оператор вида 

    $:\partial^{\vec{\mu}} \phi(0) \phi(0):$ как однородный предел при $x_E \to 0$ (т.е. любое выражение в финале усредняется по направлению $x_E$) от 
    суммы операторов

    \[
        [\partial^{\vec{\mu}} \phi(0) \phi(0)]_R = \lim_{x_E \to 0} \sum_{\vec{\nu}} { F(x_E) \partial^{\vec{\nu}} \phi(x_E) \phi(0) }
    \]
    
    Однако тут возникает сложность в том, что появятся вклады вида $\frac{1}{(\Delta |x_E|)^{2k}}$ – расходящиеся степенным образом вклады, которые приводят к смешиванию операторов разных размерностей.

    Ситуацию упрощает размерная регуляризация: в ней любой матричный элемент конечен, поэтому перенормированный оператор определяется как предел при $d \to 4$ некоторой суммы «голых» операторов. В этой схеме операторы разной размерности не смешиваются.

#### Честное вычисление (правая часть)

Теперь посмотрим, что получается, если мы найдем первую петлевую поправку [@](#eq:OPE_free) ( это средняя [диаграмма](#fig:NLO_phiphi_lambda) )

Для этого мы используем следующее свойство в размерной регуляризации:

\[
    \begin{pmatrix}
        \text{Голая функция Грина} \\
        \text{в размерной}\\
        \text{регуляризации от}
    \end{pmatrix}
    \partial_{\mu_1}\dots \partial_{\mu_n} \phi(0) \phi(0) =
    \begin{pmatrix}
        \text{убрать все члены} \\
        \text{в функии Грина от} \\
        \text{зависящие от} \; z_E
    \end{pmatrix}
    [\partial_{\mu_1}\dots \partial_{\mu_n} \phi(z_E)] \phi(0)
    \label{eq:OPE_loop_correction} 
\]

Разложение [@](#eq:OPE_free) мы можем формально написать тогда как 

\[
    :[e^{x^\mu \partial_\mu} \phi(z)] \phi(0): = \phi(z) \phi(0) + x\partial \phi(z) \phi(0) + \dots
\]

где $z \to 0$.

Подставив это в диаграмму, мы получим, аналогичное выражение [@](#eq:OPE_Euclid_final), но с $x_E$ замененным на $z_E$.

\[
    \begin{aligned}
    G'_{E}(x_E) =& -\frac{\lambda}{16\pi^2} e^{x^\mu \partial_\mu} 
    \int{d\xi \left(\frac{\mu^{2}}{4\pi\Delta^{2}}\right)^{\epsilon} \left(\frac{y'}{2}\right)^{\epsilon} 2K_{\epsilon}(y') e^{i\xi P_E z_E}} \\
    & + ( \text{убрать всё, что зависит от} \; z_E) \\
    &\quad  \text{где} \quad y'^2 = \Delta^2 z_E^2
    \end{aligned}
\]

Если убрать все зависимости от $z_E$, то, прежде всего уйдут все дробные степени, поэтому мы можем заменить

\[
    \left(\frac{y'}{2}\right)^{\epsilon} 2K_{\epsilon}(y') \to  \frac{\pi} {\sin{\pi\epsilon}} \sum_k{ \frac{(y'^2/4)^k}{k! \Gamma(k+1-\epsilon)} }
\]

И в этот момент, поскольку теперь справа всё аналитически зависит от $z_E$, формальное выражение $e^{x^\mu \partial_\mu}$ даёт сдвиг $z_E\to x_E$, и мы получаем

\[
    G'_{E}(x_E) = -\frac{\lambda}{16\pi^2}  \frac{\pi} {\sin{\pi\epsilon}}  
    \int{d\xi \left(\frac{\mu^{2}}{4\pi\Delta^{2}}\right)^{\epsilon} \sum_k{ \frac{(y^2/4)^k}{k! \Gamma(k+1-\epsilon)} e^{i\xi P_E x_E}}}
\]

И теперь перейдем к пределу при $\epsilon \to 0$. Добавим,также, члены от нулевого порядка из [диаграммы](#fig:LO_phiphi) (поскольку это первый порядок, то будет тот же ответ, что и для левой части)

\[
    \begin{split}
    &G'_{E}(x_E) =  e^{ip_{1E}x_E} + e^{ip_{2E}x_E} - \\
    &-\frac{\lambda}{16\pi^2} \int{d\xi e^{i\xi P_E x_E} \sum_k{ \frac{(y^2/4)^k}{k!^2} 
    \left( \frac{1}{\epsilon} + \ln\left(\frac{4\pi\mu^2}{\Delta^2}\right) + \psi(k+1) \right) }  }
    \end{split}
    \label{eq:OPE_loop_left_final}
\]

Если мы сравним это с выражением для левой части [@](#eq:OPE_Euclid_final_approx), то мы увидим, что они отличаются только на бесконечную константу и тот самый логарифм, который мы ожидали. Поэтому, чтобы сшить левую и правую части, нам нужно добавить

$$
    \begin{gathered}
    \text{Нужно Добавить к} \;
    G'_{E}(x_E) =
    +\frac{\lambda}{16\pi^2} \int{d\xi e^{i\xi P_E x_E} \sum_k{ \frac{(y^2/4)^k}{k!^2}} }\cdot \\
    \left( {\color{magenta} \frac{1}{\epsilon} + \ln 4\pi + \psi(1)} + 
    { \color{green} \ln \left(\frac{\mu^2 x_E^2}{4}\right) - 2\psi(k+1) + \psi(1) } \right) 
    \end{gathered}
    \label{eq:OPE_addition}
$$

Выражение в розовом цвете – это, очевидно, контрчлен перенормировки операторов вида $\partial \phi \phi$ в схеме $\bar{MS}$, а выражение в зеленом цвете – это часть коэффициентной функции.

#### Коэффициентные функции

Чтобы найти контрчлен и коэффициентную функцию к оператору 

\[
    \partial_{\mu_1} \dots \partial_{\mu_n} \phi(0) \phi(0)
    \label{eq:Operator_mu}
\]

Нужно в ряде [@](#eq:OPE_addition) вычислить коэффициент перед 

\[
    x^{\mu_1} \dots x^{\mu_n}
\]

Однако можно легко увидеть, что оператор вида [@](#eq:Operator_mu) 
дает матричные элементы в [диаграммах](#fig:LO_phiphi) выражения вида

\[
    (ip_1^{\mu_1}) \dots (ip_1^{\mu_n}) + (p_1 \leftrightarrow p_2)
\]

В то время как все контрчлены имеют вид:

\[
    (p_1+p_2)^{\mu_1} \dots (p_1+p_2)^{\mu_n} \cdots
\]

то есть они имеют перекрестные члены. Это значит, что при перенормировки оператора 
[@](#eq:OPE_addition), где производная падает только на первое слагаемое, возникнут операторы, где производные падают на все слагаемые.

---

Вычислим теперь эти коэффициентные функции в OPE, стартуя с выражение [@](#eq:OPE_addition).

- **0 порядок по $x_E$**

\[
    \Delta_{\phi^2} = \frac{\lambda}{16\pi^2} \left[
         {\color{magenta}  \frac{1}{\bar{\epsilon}} }+ 
        {\color{green} \ln \left(\frac{\mu^2 x_E^2}{4}\right)  + \gamma } 
    \right]
\]

??? info "где"

    \[
       \frac{1}{\bar{\epsilon}} = \frac{1}{\epsilon} + \ln 4\pi + \psi(1)
    \]

    – это стандартная константа, которая появляется в $\bar{MS}$-схеме. 

    и $\gamma$ – это  $\psi(1)$.

Значит, перенормированнй оператор $[\phi^2(0)]_R$ равен

\[
    [\phi^2(0)]_R = \left(1 + \frac{\lambda}{32\pi^2} \frac{1}{\bar{\epsilon}} \right) :\!\phi^2(0)\!:
\]

А коэффициентная функция для этого оператора будет

\[
    C_{\phi^2}(x) = 1 + \frac{\lambda}{32\pi^2} \left[ \ln \left(\frac{\mu^2 x_E^2}{4}\right)  + \gamma \right]    
\]

- **1 порядок по $x_E$**

Вклад будет только от первого порядк экспоненты.

\[
    \Delta( x^{\mu} \partial_{\mu} \phi(0) \phi(0) ) =  \frac{\lambda}{16\pi^2}  \int{d\xi (i \xi P_E x_E)} \left[
         {\color{magenta}  \frac{1}{\bar{\epsilon}} }+ 
        {\color{green} \ln \left(\frac{\mu^2 x_E^2}{4}\right)  + \gamma } 
    \right]
\]

Посколльку оператор $\partial \phi(0) \phi(0)$ равен $\phi(0) \partial \phi(0)$, то пробелема с перекрестными членами не возникает.

В результате, перенормированный оператор $[\partial_{\mu} \phi(0) \phi(0)]_R$ равен

\[
    [\partial_{\mu} \phi(0) \phi(0)]_R = \left(1 + \frac{\lambda}{32\pi^2} \frac{1}{\bar{\epsilon}} \right) :\!\partial_{\mu} \phi(0) \phi(0)\!:
\]

А коэффициентная функция равна

\[
    C_{\partial \phi \phi}(x) = 1 + \frac{\lambda}{32\pi^2} \left[ \ln \left(\frac{\mu^2 x_E^2}{4}\right)  + \gamma \right]
\]
